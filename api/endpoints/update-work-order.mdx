Updates an existing work order. Only work orders in DRAFT status can be modified.

## Authorizations

<ParamField header="Authorization" type="string" required>
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token
</ParamField>

## Path Parameters

<ParamField path="id" type="string" required>
  Unique work order identifier
</ParamField>

## Request Body

<ParamField body="items" type="array">
  Updated array of work order items
</ParamField>

<ParamField body="endDate" type="string">
  Updated end date in ISO 8601 format (YYYY-MM-DD)
</ParamField>

<ParamField body="locations" type="array">
  Updated array of location identifiers
</ParamField>

<ParamField body="description" type="string">
  Updated description of the work order
</ParamField>

<ParamField body="priority" type="string">
  Updated priority level: `LOW`, `MEDIUM`, `HIGH`, `URGENT`
</ParamField>

```python Python
import requests

url = "https://api.ttipl.com/v1/work-orders/wo_abc123def456"
headers = {
    "Authorization": "Bearer your_api_token_here",
    "Content-Type": "application/json"
}

payload = {
    "items": [{
        "boqId": "boq_98765xyz",
        "quantity": 200,
        "rate": 2800.00,
        "description": "Updated concrete foundation work"
    }],
    "endDate": "2024-04-15",
    "priority": "URGENT"
}

try:
    response = requests.put(url, headers=headers, json=payload)
    response.raise_for_status()
    work_order = response.json()
    print(f"Work Order Updated: {work_order['data']['id']}")
    print(f"New total amount: ${work_order['data']['totalAmount']}")
except requests.exceptions.RequestException as e:
    print(f"Error: {e}")
```
