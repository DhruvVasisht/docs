Approves a work order for execution. Changes status from PENDING_APPROVAL to APPROVED.

## Authorizations

<ParamField header="Authorization" type="string" required>
  Bearer authentication header of the form `Bearer <token>`, where `<token>` is your auth token
</ParamField>

## Path Parameters

<ParamField path="id" type="string" required>
  Unique work order identifier
</ParamField>

## Request Body

<ParamField body="approvedBy" type="string" required>
  User identifier who is approving the work order
</ParamField>

<ParamField body="comments" type="string">
  Optional approval comments or notes
</ParamField>

<RequestExample>
```bash cURL
curl -X POST "https://api.ttipl.com/v1/work-orders/wo_abc123def456/approve" \
  -H "Authorization: Bearer your_api_token_here" \
  -H "Content-Type: application/json" \
  -d '{
    "approvedBy": "user_manager123",
    "comments": "Approved for immediate execution"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.ttipl.com/v1/work-orders/wo_abc123def456/approve', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer your_api_token_here',
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    approvedBy: 'user_manager123',
    comments: 'Approved for immediate execution'
  })
});

const data = await response.json();
if (response.ok) {
  console.log('Work Order Approved:', data.data.id);
  console.log('Approved by:', data.data.approvedBy);
  console.log('Approved at:', data.data.approvedAt);
} else {
  console.error('Error:', data.error.message);
}
```

```python Python
import requests

url = "https://api.ttipl.com/v1/work-orders/wo_abc123def456/approve"
headers = {
    "Authorization": "Bearer your_api_token_here",
    "Content-Type": "application/json"
}

payload = {
    "approvedBy": "user_manager123",
    "comments": "Approved for immediate execution"
}

try:
    response = requests.post(url, headers=headers, json=payload)
    response.raise_for_status()
    approval = response.json()
    
    print(f"Work Order Approved: {approval['data']['id']}")
    print(f"Status: {approval['data']['status']}")
    print(f"Approved by: {approval['data']['approvedBy']}")
    print(f"Approved at: {approval['data']['approvedAt']}")
    
except requests.exceptions.HTTPError as e:
    error_data = response.json()
    print(f"HTTP Error: {error_data['error']['message']}")
except requests.exceptions.RequestException as e:
    print(f"Request Error: {e}")
```

<ResponseField name="success" type="boolean">
